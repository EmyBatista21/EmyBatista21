# Project Title

This project is focused on utilizing various testing tools to ensure software quality.

## Tools Used
- Cypress: A powerful tool for end-to-end testing.
- RestAssured: A popular library for testing REST services.

## ADS Course

The ADS (Análise e Desenvolvimento de Sistemas) course at IFBA provides theoretical and practical knowledge in software development.

## QA Certification

QA certification from VEM SER DBC demonstrates proficiency and knowledge in quality assurance practices.
